##### Installation:

- *(Probably create a virtual environment)*
- Find the wheel appropriate wheel for pyaudio [here](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio).
- `pip install wheel.whl`
- `pip install requirements.txt`

##### Dependencies:

- `pyaudio`
- `synthesizer`
- `colour`
